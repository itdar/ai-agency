## 단계
1. 변경된 파일 목록 확인
2. 루트 AGENTS.md 글로벌 규약 준수 여부 점검
3. 체크리스트 기반 리뷰:
   - 셸 스크립트: `set -euo pipefail` 존재, 변수 인용 처리, shellcheck 통과
   - src/HOW_TO_AGENTS.md: 플레이스홀더 형식 일관성(`{snake_case}`), 템플릿 구조 보존
   - 번역 파일: 원본과 구조 동일, 코드블록 미번역
4. 벤더 중립성 위반 여부 확인 (특정 AI 도구 종속 코드 없는지)
5. 비추론 가능 원칙 준수 확인 (추론 가능한 내용이 AGENTS.md에 포함되지 않았는지)
