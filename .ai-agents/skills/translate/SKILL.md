# 스킬: 번역 워크플로우

## 트리거
영문 원본(README.md, src/HOW_TO_AGENTS.md)이 변경되어 번역 파일 동기화가 필요할 때

## 단계
1. 영문 원본의 변경 내역 확인 (`git diff`)
2. 변경된 섹션을 각 언어 번역 파일에서 찾기
3. 번역 수행
   - 코드 블록, CLI 명령어, 파일 경로는 번역하지 않음
   - ASCII 다이어그램 정렬을 각 언어 문자 폭에 맞게 조정
4. 전체 번역 파일의 구조가 영문 원본과 동일한지 검증

## 완료 조건
- 모든 10개 언어 파일이 영문 원본과 구조 동일
- 코드 블록이 번역되지 않음
- 다이어그램 정렬 정상
